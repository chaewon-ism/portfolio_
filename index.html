탄탄한 기본기를 바탕으로 하나씩 채워 나갑니다
유지보수 하기에 효율적인 코드, 이용하기에 접근성이 좋은 코드 를 위해 항상 스스로 발전하며 채워 나가겠습니다

  • HTML / CSS

    기본적인 틀과 구조를 잡을 줄 알며, 웹표준에 적합한 태그와 코드 사용으로 반응형 및 적응형 제작이 가능합니다

    css를 통해 정적인 요소를 꾸미고 동적인 요소를 구현 할 수 있습니다

  • Jquery / script

    메뉴 구성, 팝업, 슬라이드, 아코디언과 같은 간단한 script는 직접 코드 작성이 가능합니다

    기본적인 문법을 읽을 줄 알며, 오픈 소스를 적절히 활용 할 수 있습니다

  • Photoshop / Illust / XD

    전반적인 툴의 기능을 알고 활용 가능하며 와이어 프레임 및 디자인 시안을 만들 수 있습니다

    사진 보정, 로고 제작, 포스터 및 배너 제작이 가능합니다

  • PPT / Word / Xcel / HWP

    진행 하는 프로젝트 및 업무에 필요한 문서 서류를 작성 할 수 있습니다.

  • 웹디자인 기능사 자격증 취득 (2022.03.11)
  • 수용하는 자세를 가지고 있으며 원활한 의사소통을 위해 노력합니다
    항상 스스로의 부족함을 채우려 하고,
    언제나 귀 기울이는 자세가 준비 되어있습니다 :)
